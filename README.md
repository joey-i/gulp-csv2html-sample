# gulp-csv2html-sample
このサンプルは、京都市で提供されている観光情報のCSVを元にして、  
gulp-ejsとcsv-parserを使用してページ生成をするサンプルファイルです。

## サンプル使用方法

レポジトリーをクローン後、以下のコマンドを実行すると依存パッケージがインストールされます。

```
npm install
```


### ページ生成コマンド

```
npx gulp ejs
```

### 確認用サーバー起動
http://localhost:3000 でページを確認することができます。

```
npm run http-server
```
